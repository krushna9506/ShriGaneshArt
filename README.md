# Ganesh Arts Management System

This starter workspace includes:
- React + Vite + Tailwind frontend
- Express.js backend with JWT auth and REST endpoints
- Supabase-ready schema and env templates

## Run
1. npm install --prefix frontend
2. npm install --prefix backend
3. npm run dev

Frontend: http://localhost:3000
Backend: http://localhost:4000

Use admin / admin123 to test the JWT login.
